Heading sizes start from 1, being the largest, to 6, being the smallest. Browsers will normally add a space after the title, no need to break the line - br. Headings are important because search engines use the headings to index the structure and content of your web pages.

Each HTML heading has a default size. You can specify thw size for any heading with the style attribute, using the CSS font-size property.

pre Element

The HTML pre element defines preformatted text. Text inside the element is displayed in a fixed-width (usually Courier), and it preserves both spaces and line breaks:

   My Bonnie lies over the ocean.

   My Bonnie lies over the sea.

   My Bonnie lies over the ocean.
   
   Oh, bring back my Bonnie to me.

Paragraphs

The browser will remove any extra spaces and extra lines automatically in the paragraph element. Most browsers will display HTML paragraphs correctly if you forget the end tag.

The HTML head Element

The HTML head element is a container for metadata. HTML metadata is data about the HTML document. Metadata is not displayed.

The hr tag defines a thematic break in an HTML page, and is displayed a horizontal rule.

      • Quotation and Citation Elements

      For short quotations you can use the element "q" and close with "/q":

      WWF's goal is to: Build a future where people live in harmony with nature.

      By using "blockquote cite="html for website" you can quote and cite a website:

      Here is a quote from WWF's website:

      For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
